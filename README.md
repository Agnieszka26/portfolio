# Portfolio – Agnieszka26

Responsywne portfolio frontend developerki, zbudowane w oparciu o **React**, **Next.js** i **Sanity** jako CMS. Projekt przedstawia moje umiejętności, doświadczenie i wybrane projekty.

🌐 **Live preview:** [portfolio-agnieszka26.vercel.app](https://portfolio-agnieszka26.vercel.app/)

---

## 🛠 Stack technologiczny

- **React** / **Next.js 14** (App Router)
- **Sanity** (CMS + Studio pod `/studio`)
- **SCSS** (modułowy styl, bez frameworków UI)
- **Vercel** (hosting)

---

## ✨ Funkcje

- W pełni responsywny layout (RWD)
- Dynamiczne ładowanie projektów z Sanity
- Podstrony:
  - **Home** – powitanie i krótki opis
  - **Projekty** – lista projektów
  - **Kontakt** – dane kontaktowe
- Edycja treści w Sanity Studio (w tym Markdown dla case study)
- Czysty, minimalistyczny design

---

## 🚀 Jak uruchomić lokalnie

1. Sklonuj repozytorium:
   ```bash
   git clone https://github.com/Agnieszka26/portfolio.git
   cd portfolio
   ```

2. Zainstaluj zależności:
   ```bash
   npm install
   ```

3. Skonfiguruj zmienne środowiskowe w `.env.local` (Sanity project ID, dataset, API version).
`
NEXT_PUBLIC_SANITY_DATASET=
NEXT_PUBLIC_SANITY_PROJECT_ID=
`


4. Uruchom projekt:
   ```bash
   npm run dev
   ```

Studio: [http://localhost:3000/studio](http://localhost:3000/studio)
